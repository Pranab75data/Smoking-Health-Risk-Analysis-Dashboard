# Smoking-Health-Risk-Analysis-Dashboard
This interactive Power BI dashboard provides a comprehensive analysis of the health risks associated with smoking across multiple body organs. It enables users to compare healthy vs. damaged conditions and understand how smoking behavior impacts different demographic and health indicators.  Key Features &amp; Insights.

## Key Features & Insights:

* Displays total patient count, along with average age and BMI, dynamically updated based on filters.

* Visualizes smoking status distribution (Never, Current, Former smokers) using intuitive donut charts.

* Analyzes smoking patterns through trends in years of smoking (YOS) and cigarettes per day (CPD) across age groups.

* Compares smoking status by gender, highlighting behavioral differences between male and female patients.

* Assesses cholesterol and hypertension risk levels (High, Normal, Low) segmented by age groups.

* Provides organ-wise health analysis (Heart, Lungs, Liver, Kidney, and Human anatomy) with visual indicators showing healthy and damaged conditions.

* Includes interactive slicers and navigation buttons for seamless exploration and focused analysis.
